## Api

| 参数    |       描述       |  类型   | 可选值 | 必须 | 默认值      |
| ------- | :--------------: | :-----: | :----: | :--: | ----------- |
| visible |   是否显示预览   | Boolean |        |  是  | false       |
| list    |     图片列表     |  Array  |        |  是  | [{url: ''}] |
| index   | 当前展示图片索引 | Number  |        |  是  | 0           |
