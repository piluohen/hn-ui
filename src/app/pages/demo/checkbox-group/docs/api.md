## Api

| 参数     |    描述     |  类型   | 可选值 | 必须 | 默认值 |
| -------- | :---------: | :-----: | :----: | :--: | ------ |
| options  |  全部数据   |  Array  |        |  是  | -      |
| ngModel  |   指定值    |  Array  |        |  是  | -      |
| disabled | 是否 d 禁用 | Boolean |        |  否  | false  |

## Event

| 名称          |         描述         | 返回值        |
| ------------- | :------------------: | ------------- |
| ngModelChange | 选中数据变化时的回调 | 指定值[value] |
