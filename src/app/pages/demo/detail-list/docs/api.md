## Api

| 参数    |           描述           |  类型  | 可选值 | 必须  | 默认值 |
| ------- | :----------------------: | :----: | :----: | :---: | ------ |
| columns |         数据列表         | Array  |        | true  | []     |
| data    |         数据对象         | Object |        | true  | {}     |
| colon   | 展示标题与内容之间的符号 | String |        | false | ：     |
| render  |         模板对象         | Object |        | false | {}     |
