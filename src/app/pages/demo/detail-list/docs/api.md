## Api

| 参数   |           描述           |  类型  |         可选值         | 必须 | 默认值     |
| ------ | :----------------------: | :----: | :--------------------: | :--: | ---------- |
| mode   |           模式           | String | horizontal \| vertical |  否  | horizontal |
| list   |         数据列表         | Array  |                        |  是  | []         |
| data   |         数据对象         | Object |                        |  是  | {}         |
| colon  | 展示标题与内容之间的符号 | String |                        |  否  | ：         |
| render |         模板对象         | Object |                        |  否  | {}         |
