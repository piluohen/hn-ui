# Hn-ui

基于 Angular7、ng-zorro-antd7.5.1 拓展的组件库。
目的：让开发更简单

## 文档及 demo

[https://piluohen.github.io/hn-ui/](https://piluohen.github.io/hn-ui/)

## 安装使用

```bash

# 安装
npm install hn-ui --save

# 使用
import { HnUiModule } from 'hn-ui';

@NgModule({
  imports: [
    CommonModule,
    HnUiModule,
  ],
  declarations: [],
  providers: [],
  exports: []
})

```
